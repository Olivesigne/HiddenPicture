🔐 Débloquer les sources inconnues via Family Link

Si l’appareil de votre enfant est supervisé, suivez ces étapes pour autoriser l’installation.

📌 Pourquoi cette étape est nécessaire ?

Sur les appareils supervisés par Google Family Link, l’option "Installer des applications inconnues" est désactivée par défaut. Vous devez l’activer depuis le téléphone du parent.

📖 Instructions pour les appareils supervisés

📱 Depuis le téléphone du parent :

  1. Ouvrez l’application Google Family Link 📱
  2. Sélectionnez votre enfant dans la liste
  3. Cliquer sur l'appareil de votre enfant ⚙️
  4. Activez applications de sources inconnues ✅
  5. Cliquer sur ok pour finaliser l'action
  6. Finissez le tuto précédent

✅ Installation réussie !

Vous pouvez maintenant profiter du jeu sans problème. 🎉
