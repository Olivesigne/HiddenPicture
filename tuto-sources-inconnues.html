🔐 Autoriser les applications de sources inconnues

Suivez ces étapes pour installer Hidden Picture sur l'appareil de votre enfant.

📌 Pourquoi cette étape est nécessaire ?

Par défaut, Android bloque l’installation d’applications qui ne viennent pas du Google Play Store. Comme Hidden Picture est un jeu indépendant, vous devez activer cette option temporairement.

📖 Instructions selon votre version d'Android *

📱 Android 10 et plus

  1. Ouvrez Paramètres ⚙️
  2. Allez dans Applications et notifications
  3. Appuyez sur Accès spécial aux applications
  4. Sélectionnez Installer des applications inconnues
  5. Choisissez l’application qui sert à télécharger le fichier (Chrome, Samsung Internet, etc.)
  6. Activez Autoriser cette source ✅
  7. Si vous avez un message type : "Désactivé par l'administrateur", veuillez désactiver cette restriction dans l'application du gérant de cet appareil :
    8. Accéder au tuto Family Link
  8. Installez Hidden Picture
  9. Une fois terminé, désactivez l'option pour plus de sécurité.

📱 Android 8 et 9

  1. Ouvrez Paramètres ⚙️
  2. Allez dans Sécurité et confidentialité
  3. Appuyez sur Sources inconnues
  4. Activez l’option ✅
  5. Installez Hidden Picture
  6. Une fois terminé, désactivez l'option pour la sécurité.

📱 Android 7 et versions antérieures

  1. Ouvrez Paramètres ⚙️
  2. Allez dans Sécurité 🔐
  3. Cochez Sources inconnues ✅

Incompatible à présent

  1. Acceptez l’avertissement
  2. Installez Hidden Picture
  3. Une fois terminé, décochez l'option pour la sécurité.

📱 *Trouver sa version d'Android

  1. Ouvrez Paramètres ⚙️
  2. Allez dans Informations systèmes 🔐
  3. Trouver Version android ou Version de build android ✅
  4. Installez Hidden Picture d'après les étapes ci-dessus

✅ Installation réussie !

Une fois l'application installée, vous pouvez jouer sans problème. Amusez-vous bien ! 🎉
