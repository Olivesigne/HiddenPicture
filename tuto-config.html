📊 Vérifier les spécifications de son téléphone

Découvrez comment voir la version d’Android, la RAM, le processeur et l’espace de stockage de votre appareil.

📌 Trouver la version d'Android

  1. Ouvrez Paramètres ⚙️
  2. Allez dans À propos du téléphone 📱
  3. Recherchez la ligne Version Android
  4. Votre version d’Android s'affiche ici ✅

💾 Vérifier la RAM et le processeur

  1. Ouvrez Paramètres ⚙️
  2. Allez dans À propos du téléphone
  3. Cherchez Informations matérielles ou Caractéristiques techniques
  4. Vous verrez :
    • RAM : Indique la mémoire vive de votre téléphone
    • Processeur : Le modèle et la vitesse du CPU

📂 Vérifier l’espace de stockage

  1. Ouvrez Paramètres ⚙️
  2. Allez dans Stockage 📂
  3. Vous verrez l’espace total et l’espace libre sur votre téléphone ✅

🔍 Applications tierces (optionnel)

Si votre téléphone ne montre pas toutes les infos, vous pouvez utiliser une application comme CPU-Z ou AIDA64 pour obtenir des détails précis.

✅ Vérification terminée

Si votre appareil correspond à la configuration minimale, vous pouvez télécharger et jouer à Hidden Picture ! 🎉
