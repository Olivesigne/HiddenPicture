🔑 Avant de lancer Hidden Picture

Avant de lancer Hidden Picture, assurez-vous d’accorder les autorisations suivantes pour un bon fonctionnement.

📸 Autoriser l’accès à l’appareil photo

  1. Ouvrez Paramètres ⚙️
  2. Allez dans Applications 📱
  3. Recherchez Hidden Picture et appuyez dessus
  4. Allez dans Autorisations 🔓
  5. Activez Appareil photo 📸

📍 Autoriser l’accès à la localisation

  1. Allez dans Paramètres ⚙️
  2. Ouvrez Applications 📱 puis sélectionnez Hidden Picture
  3. Allez dans Autorisations 🔓
  4. Activez Localisation 📍 et sélectionnez Autoriser tout le temps

⚠️ Cette autorisation est essentielle pour suivre les joueurs sur la carte.

🎧 Autoriser l’accès à la musique et l'audio

  1. Ouvrez Paramètres ⚙️
  2. Allez dans Applications 📱 et trouvez Hidden Picture
  3. Allez dans Autorisations 🔓
  4. Activez Stockage 📂

Cette autorisation est nécessaire pour enregistrer les photos prises pendant le jeu.

🎯 Vous êtes prêt !

Si toutes les autorisations sont activées, vous pouvez profiter de Hidden Picture sans problème !
